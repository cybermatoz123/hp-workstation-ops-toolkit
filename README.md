\# HP Workstation Ops Toolkit



PowerShell-based workstation operations toolkit designed for system auditing, endpoint visibility, diagnostics, cleanup automation, reporting, and scheduled maintenance.



\---



\# Overview



This project was built to improve workstation management, operational visibility, and IT automation workflows on a Windows-based production workstation.



The toolkit automates:



\* System inventory collection

\* Startup application auditing

\* Disk usage reporting

\* Downloads cleanup

\* Network diagnostics

\* System health monitoring

\* HTML dashboard generation

\* Scheduled audit execution
