\# HP Workstation Ops Toolkit



A PowerShell-based workstation operations toolkit built for auditing, cleanup, diagnostics, reporting, and basic endpoint visibility.



\## Purpose



This project helps automate workstation management tasks including:



\- System inventory

\- Startup application auditing

\- Disk usage reporting

\- Downloads cleanup

\- Network diagnostics

\- System health checks

\- HTML dashboard reporting

\- Scheduled weekly audits



\## Folder Structure



/scripts  

PowerShell automation scripts.



/reports  

Generated HTML/text reports.



/screenshots  

Dashboard screenshots and visual documentation.
